Gales Gas Sales

2013 – Present
Golang, MongoDB, NodeJS, React

Gales is a gas and fuel company in the Niagara Region of ON. CA. In 2013 I was presented with the task of modernizing their data collection for the 16 locations currently in operation. After a few iterations using Filemaker I proposed a web based approach and we embarked on the development of two applications and several micro-services.

  • SPA with React Redux UI
  • Hapi NodeJS API
  • Golang Docker Micro-services
  • MongoDB NoSQL DB

Internal IaC Project

2016 – Present
AWS, Terraform

After deciding to update my server infastructure utilizing IaC (Infastructure as Code) I did some research and settled on Terraform as the tool. The switch from manually firing up and configuring AWS EC2 instances to using a sophisticated tool like Terraform required an investment in time but certainly well worth the effort even for a fairly small infastructure. Now I'm able to test all my changes in a test environment before deploying to production knowing I won't break and legacy applications are now issolated and can be upgraded without concern for how other apps might be affected.

Niagara Prehung Doors

2013 – 2015
AngularJS, Grape, Golang, MongoDB

Niagara Prehung Doors is a manufacturer and distributer for residential custom exterior doors. Initially I created a Filemaker application to provides quotes and manage sales. As requirements grew so did the limits of Filemaker. A decision was made to convert the project to a web based application so that we could take advantage of all that a web application has to offer. Features like securely sending PDF quotes that could be revised and versioned as their customers made changes to their selections was made possible.

  • SPA with Angular UI
  • Ruby Grape API
  • MongoDB NoSQL DB
  • Golang Google API to interface with Google Calendar
  • Golang AWS SDK to interface with the AWS SES and S3 services
  • Golang gofpdf library to create PDF files

The Teachable Project

2012 – 2013
AngularJS, Zend Framework

The Teachable Project is a fully bilingual site commissioned by the Ontario Educational Services Corporation to educate teachers and students on Accessibility Awareness.

A custom CMS was built to allow the content providers to enter data with a simple to use interface, uploading image files and converting markdown to HTML.
