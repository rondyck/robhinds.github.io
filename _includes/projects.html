Clients & Projects

CBT Niagara

2017 (6 wks)
Apollo Graphql, Golang, MongoDB, NodeJS, React

CBT Niagara is a cognitive-behaviour therapy clinic. I was approached by them to provide a simpler and easy to use patient scheduling solution than the larger medical institution based ones available. I created an SPA utilizing Material Design components to fulfill their requirements. The receptionist/secretary is a much happier person as the tedious work of transferring data from paper forms is now eliminated and the practitioners enter their respective data into the application.

  • SPA with React Redux UI
  • Apollo Graphql client
  • Hapi NodeJS Graphql server
  • MongoDB NoSQL DB

Universal Windows

2016 – Present
AWS Lambda, Golang, MongoDB, NodeJS, React Native

Universal Windows is a local window replacement company. Much of the owners time is spent meeting with home owners and creating quotes. To facilitate this effort I created a web based React Native application for the owners iPad. The owner is exceptionally happy with the level of detail he is able to record eliminating costly mistakes when missed. Potential customers receive a PDF quote via email that is versioned so that both parties can track changes.

  • React Native IOS App
  • Hapi NodeJS API
  • Golang written AWS Lambda Functions deployed with Apex
  • MongoDB NoSQL DB

Internal IaC Project

2016 – Present
AWS, Terraform

After deciding to update my server infrastructure utilizing IaC (Infrastructural as Code) I did some research and settled on Terraform as the tool. The switch from manually firing up and configuring AWS EC2 instances to using a sophisticated tool like Terraform required an investment in time but certainly well worth the effort even for a fairly small infrastructure. Now I'm able to test all my changes in an isolated test environment before deploying to production knowing it won't break. Legacy applications are also well isolated and can be upgraded without concern for how other apps might be affected.

  • Terraform
  • Amazon Web Services

Niagara Prehung Doors

2013 – 2015
AngularJS, Grape, Golang, MongoDB

Niagara Prehung Doors is a manufacturer and distributer of residential custom exterior doors. Initially I created a Filemaker application to provide quotes and manage sales. As requirements grew so did the limits of Filemaker. A decision was made to convert the project to a web based application so that we could take advantage of all that a web application has to offer. Features like securely sending PDF quotes that can be revised and versioned as customers make changes to their selections is now possible.

  • SPA with Angular UI
  • Ruby Grape API
  • MongoDB NoSQL DB
  • Golang Google API to interface with Google Calendar
  • Golang AWS SDK to interface with the AWS SES and S3 services
  • Golang gofpdf library to create PDF files
